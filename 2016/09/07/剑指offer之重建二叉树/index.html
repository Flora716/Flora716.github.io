剑指offer之重建二叉树

发表于 2016-09-07

题目： 输入某二叉树的前序遍历和中序遍历的结果，请重建出该二叉树。假设输入的前序遍历和中序遍历的结果中都不含重复的数字。例如输入前序遍历序列{1,2,4,7,3,5,6,8}和中序遍历序列{4,7,2,1,5,3,8,6}，则重建二叉树并返回。

解题思路：

  1. 先序第一个节点是根节点node
  2. 把根节店带入到中序遍历中，确定位置为p，确定左右子树的中序遍历
  3. 先序遍历第二个位置到p，即node左子树的先序遍历，剩下的就是node右子树的先序遍历

  4. 找出四个数组，递归

    public class Solution {
 public TreeNode reConstructBinaryTree(int [] pre,int [] in) {

     TreeNode root = new TreeNode(pre[0]);  //前序的第一个节点为根节点

     int len = pre.length;
     //当只有一个节点时
     if(len == 1){
         root.left=null;
         root.right=null;
         return root;
     }

    int rootval = root.val;
    int i;
    for(i =0;i<len;i++){
        if(rootval == in[i])
            break;
    }

    //创建左子树
    if(i>0){
        int[] pr = new int[i];
        int[] ino = new int[i];
        for(int j=0;j<i;j++){
            pr[j]=pre[j+1];   //左子树的先序遍历
        }
        for(int j=0;j<i;j++){
            ino[j]=in[j];    //左子树的中序遍历
        }
        root.left = reConstructBinaryTree(pr,ino);
    }else{
        root.left = null;
    }
    //创建右子树
    if(len-i-1>0){
        int[] pr = new int[len-i-1];
        int[] ino = new int[len-i-1];
        for(int j=i+1;j<len;j++){
            pr[j-i-1]=pre[j];    //右子树的先序遍历
            ino[j-i-1]=in[j];     //右子树的中序遍历
        }
        root.right = reConstructBinaryTree(pr,ino);
    }else{
        root.right = null;
    }
    return root;
}

}
