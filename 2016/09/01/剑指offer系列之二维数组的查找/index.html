剑指offer系列之二维数组的查找

发表于 2016-09-01

题目： 在一个二维数组中，每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

基本思想： 因为二维数组从左到右从上到下都是有序的，
所以从右上角开始遍历，
若右上角的数小于要找的值，则向下一行寻找，行+1，
若右上角的数大于要找的值，则向该行的前一列寻找，即列-1。

代码部分：

public class findFromMatrix2 {
public static boolean Find(int [][] array, int num){
    boolean found = false;
    int row = 0;
    int col = array[0].length -1;
    if(array != null && row > 0 && col >0){
        while(row < array.length - 1 && col >= 0){
            if(num == array[row][col]){
                found = true;
            }else if(num > array[row][col]){
                ++row;
            }else{
                --col;
            }
        }
    }
    return found;
}

public static void main(String[] args){
    int arrays[][] = {{1,2,8,9},{2,4,9,12},{4,7,10,13},{6,8,11,15}};
    if(Find(arrays,5)){
        System.out.println("二维数组中存在这样的一个数！");
    }else{
        System.out.println("二维数组中不存在这样的一个数！");
    }
  }
}
